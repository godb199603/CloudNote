

##功能：

- 实现了登录、注册、保持登录。
- 实现了笔记的增删改查。
- 基于Bmob后端云，笔记可以在云端与本地之间同步。
- 使用RxJava处理诸如数据库操作、数据同步等异步任务。
- 使用GreenDAO处理数据的本地持久化操作。
